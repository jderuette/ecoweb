Comment ça marche ?

Quatre étapes simples pour votre site web écoconçu

étape1

Téléchargez WebLeger

Récupérez l'outil de développement et le dossier modèle pour démarrer rapidement en faisant un fork depuis gitHub

étape2

Créez votre contenu

Rédigez vos pages en Markdown et ajoutez vos images, prsonalisez les contenus grâce aux entêtes MarkDown, pesonalisez les couleurs sans faire de CSS. Aucune base de données nécessaire

étape3

Générez votre site

JBake transforme vos contenus en un site HTML statique et Webleger le rend ultra-léger et performant

étape4

Hébergez simplement

Déployez sur GitHub Pages, un serveur Apache/Nginx ou notre hébergement dédié
