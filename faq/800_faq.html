La Foire aux questions pour répondre à presque toutes vous questions.

Peut-on créer une boutique en ligne avec WebLege ?

Pas vraiment, il est possible d'exposer vos produits et service, de les présenter. Cependant pour les payer il faudra utiliser une application externe.
Par exemple Strippe permet d'acheter un produit avec le système des liens de paiement.

WebLege est-il payant ?

Non WebLege est gratuit. WebLege utilise des outils OpenSource et déja gratuits.
Vous pouvez soutenir le developpeur qui agrege tout ces outils ou soutenir directement les projets OpenSource utilisé par WebLege.
Vous pouvez passer par l'agence EthikNet pour faire réaliser votre site web ou obtenir del'aide pour le faire vous même.

Comment sera créé mon site ?

La création d'un site web classique nécéssite :

  1. une direction artistique
  2. des contenus
    1. textuel
    2. éventuellement des images

Tous ces éléments peuvent être disponibles au début de votre projet ou pas. Ils peuvent aussi évoluer (notament les contenus).

Webleger encourage un fonctionnement itératif donc enchainer ces 3 étapes sur des petits bout du site.

Pour WebLeger il faut :

  • héberger le code source
  • installer les outils de création sur votre ordinateur
  • puis enchainer avec les 3 étapes précédentes aussi souvent que nécéssaire
  • vérifer votre site un test (puis le corriger)
  • le livrer sur le serveur Web qui hèbergera le site web publique.
